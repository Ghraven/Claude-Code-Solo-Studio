# /help — List All Commands

Print this exactly:

```
LEAN GAME STUDIO — COMMANDS
────────────────────────────

GETTING STARTED
  /setup          Set up your game (engine, platform, genre) and generate starter files

BUILDING
  /new-feature    Start building a new game feature
  /level-design   Design a level or scene layout
  /ui-design      Design a menu, HUD, or screen

QUALITY
  /code-review    Review code for bugs and issues
  /qa-check       Full quality check before moving on
  /bug-report     Log and fix a bug

RELEASING
  /build          Export/build checklist for your platform

TRACKING
  /status         See current sprint and project state
  /help           Show this list

AGENTS (invoke directly by name for focused work)
  game-designer   Mechanics, balancing, GDD updates
  programmer      All code — Godot GDScript or Unity C#
  level-designer  Level layouts and scene structure
  ui-designer     Menus, HUD, screen transitions
  qa              Code review and testing
  release         Build, export, versioning

TIP: Most of the time you don't need to use agents directly.
Just use the slash commands above — they handle everything.
Agents are there if you want deep focused work on one domain.
```
