# /ui-design — Design a Screen or HUD Element

Read CLAUDE.md first for platform, engine, and genre.

Ask: "What UI element or screen do you need? (examples: main menu, pause screen, HUD, game over screen, settings)"

Then produce a complete UI design using the ui-designer agent format:
- ASCII layout sketch
- Element details (position, size, behavior)
- Transitions (enter/exit animations)
- Mobile considerations (tap zones, finger-friendly sizes)
- Engine-specific node/component list

End with: "Want me to write the code for this UI now?"
