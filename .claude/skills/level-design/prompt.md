# /level-design — Design a Level or Scene

Read CLAUDE.md first for genre, platform, and engine.

Ask: "Which level are you designing? (number/name) And is this a new level or refining an existing one?"

Then produce a complete level design document using the level-designer agent format:
- Level goal and estimated time
- ASCII layout sketch
- Element placement (enemies, items, hazards, exit)
- Teaching moment (what mechanic this level reinforces)
- Scene structure for the engine (nodes for Godot, GameObjects for Unity)

End with: "Want me to turn this into actual scene files now?"
